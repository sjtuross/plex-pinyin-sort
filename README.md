# plex-pinyin-sort

plex拼音排序脚本
plex的中文电影默认排序是按照笔画数排的，对检索中文电影十分不友好，运行此脚本后可对指定电影库进行拼音排序，并可使用拼音首字母检索电影。

# 需要安装plexapi

pip install plexapi

更多PLEX中文插件请访问plexmedia.cn
